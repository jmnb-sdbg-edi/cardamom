Welcome to the CDMW DEMOS
=========================

The purpose of CDMW DEMOS is to provide comprehensive examples for writing
software based on CDMW runtime middleware. Provided Makefiles may be reused
to write more sofisticated ones that suit needs of CDMW users.

To build and run demos refer to README.txt file in each demo directories
For java demos don't forget to have CLASSPATH set with installed java libs and path to jacorb.properties file

All the demos are installed, but some of them can be build and executed only
if a particular option/tool was installed and selected in the configure.

For the following demos, splice need to be installed and selected in the configure:
demo_splice
demo_ft_dcps
demo_ccm_ft
